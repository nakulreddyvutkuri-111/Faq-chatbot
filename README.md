# 🤖 FAQ Chatbot

## 📌 Overview

The FAQ Chatbot is a web-based application developed using HTML, CSS, and JavaScript. It provides automated responses to frequently asked questions by matching user queries with a predefined FAQ dataset.

The chatbot uses basic Natural Language Processing (NLP) techniques such as text preprocessing, tokenization, and similarity matching to identify the most relevant question and return the corresponding answer.

## ⚙️ How It Works

1. User enters a question in the chat input.
2. The chatbot preprocesses the text.
3. The system compares the user's question with stored FAQs.
4. Similarity matching is performed.
5. The most relevant FAQ answer is displayed.
6. If no suitable match is found, the chatbot provides a fallback response.
