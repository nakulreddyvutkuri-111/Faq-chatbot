FAQ Chatbot

The FAQ Chatbot is a web-based application built using HTML, CSS, and JavaScript. It is designed to provide automated responses to frequently asked questions by matching user queries with a predefined FAQ database.

The chatbot uses text preprocessing techniques such as tokenization and lowercase normalization, along with similarity matching algorithms, to identify the most relevant question and return the corresponding answer. The application features a modern chat-based user interface that enables users to interact with the chatbot in a simple and intuitive manner.

Key Features:
• Interactive chatbot interface
• FAQ-based automated responses
• NLP-inspired text preprocessing
• Similarity matching for question recognition
• Responsive and user-friendly design
• Enter key support for quick interactions

Technologies Used:
• HTML
• CSS
• JavaScript

This project demonstrates fundamental concepts of chatbot development, Natural Language Processing (NLP), text similarity matching, DOM manipulation, and front-end web development.
